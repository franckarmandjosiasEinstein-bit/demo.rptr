Réponse à la question

On utilisera le format png parce que le format JPEG ne supporte pas la transparence (il utilise un fond opaque), tandis que le PNG permet une couche alpha pour la transparence, ce qui est essentiel pour un logo devant se fondre sur différents arrière-plans.
