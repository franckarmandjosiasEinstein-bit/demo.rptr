Propriétés CSS essentielles pour les tableaux

border-collapse

table { border-collapse: collapse; }

☑ La propriété border-collapse: collapse supprime l'espace entre les bordures des cellules adjacentes pour un rendu plus propre.

caption

La balise <caption> définit un titre ou une description pour le tableau :

caption { font-style: italic; color: #555; margin-bottom: 8px; }

thead, tbody, tfoot

Ces balises structurent le tableau en trois sections :

  • <thead> : L'en-tête du tableau (une seule ligne généralement)
  • <tbody> : Le corps du tableau (contient les données)
  • <tfoot> : Le pied du tableau (résumés, totaux, moyennes)

Styles pour les sections du tableau

th { background-color: #2E74B5; color: white; padding: 10px; }

tbody tr:hover { background-color: #F0F0F0; }

tfoot { background-color: #E2EFDA; font-weight: bold; }

Bonnes pratiques pour styliser les tableaux

✓ Recommandations

  • Utilisez border-collapse: collapse pour un rendu net
  • Distinguez visuellement l'en-tête avec une couleur et une police différente
  • Appliquez :hover aux lignes du corps pour améliorer la lisibilité
  • Utilisez une couleur différente pour le pied du tableau si nécessaire
  • Assurez-vous que le contraste couleur/texte est suffisant
  • Ajoutez du padding pour aérer visuellement les cellules
  • Alignez les nombres à droite pour une meilleure compréhension
