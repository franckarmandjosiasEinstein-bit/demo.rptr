Styles Internes avec la Balise <style>

Cette page démontre l'utilisation d'une feuille de style interne. Tous les styles sont définis dans la balise <style> du <head>, ce qui permet d'appliquer les mêmes règles à tous les éléments corresponding de la page.

Avantages de cette méthode :

✓ L'indication de style dans <style> porte sur tous les éléments correspondants de la page, contrairement à la méthode inline.

✓ Les styles sont centralisés dans le <head>

✓ Plus facile à maintenir qu'inline

Inconvénients :

✗ Les styles ne sont pas partagés entre plusieurs pages

✗ Augmente la taille du fichier HTML
