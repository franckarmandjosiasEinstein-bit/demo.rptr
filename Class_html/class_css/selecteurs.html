Différences entre les sélecteurs :

Sélecteur Symbole Usage Occurrences
id # Cibler un élément unique Une seule fois par page
class . Cibler plusieurs éléments Autant de fois que voulu
Élément simple (aucun) Cibler tous les éléments du type Tous les éléments correspondants
Contextuel (espace) Cibler un élément dans un contexte Selon la position dans le DOM

Points clés à retenir :

✓ Les ids (#) doivent être uniques dans une page et s'utilisent pour identifier un élément particulier.

✓ Les classes (.)​ peuvent être utilisées autant de fois qu'on le souhaite et permettent de grouper des élément avec le même style.

✓ La spécificité : Un sélecteur id a une spécificité plus élevée qu'une classe, qui a elle-même une spécificité plus élevée qu'un sélecteur d'élément.
