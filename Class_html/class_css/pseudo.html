Récapitulatif

États des liens (Pseudo-classes) :

  • :link - Lien non visité
  • :visited - Lien déjà visité
  • :hover - Au passage de la souris
  • :active - Pendant le clic

Pseudo-éléments typographiques :

  • :first-letter - Première lettre d'un élément
  • :first-line - Première ligne d'un élément
