Unités de mesure en CSS

1. Unités absolues

Pixels (px)

Cette police fait 30px - Unité semi-absolue basée sur les pixels de l'écran

Les pixels sont l'unité la plus couramment utilisée pour les bordures et les espacements précis.

h1 { font-size: 30px; }

Points (pt)

Cette police fait 18pt - Utilisée principalement pour l'impression

Les points (1/72 pouce) sont historiquement utilisés pour les documents imprimés.

h2 { font-size: 18pt; }

2. Unités relatives à la police

Em (em)

Cette police fait 1.5em (1.5 × 16px = 24px)

L'unité em est relative à la taille de la police du parent. Si le parent a 16px, 1.5em = 24px.

p { font-size: 1.5em; } /* 1.5 × taille du parent */

Rem (rem - Root Em)

Cette police fait 2rem (2 × 16px du html = 32px)

L'unité rem est relative à la taille de la police de la racine (<html>), pas du parent direct. Elle offre plus de prévisibilité.

p { font-size: 2rem; } /* 2 × 16px racine */

💡 Avantage du rem : Tous les éléments avec rem changent proportionnellement si vous modifiez la taille de base du html.

3. Unités relatives au viewport

Viewport Width (vw) et Viewport Height (vh)

Cet élément fait 100vw × 50vh (100% largeur × 50% hauteur de l'écran)

Ces unités sont parfaites pour les mises en page full-screen. 1vw = 1% de la largeur de l'écran, 1vh = 1% de la hauteur.

.plein-ecran { width: 100vw; height: 50vh; }

4. Pourcentage (%)

Largeur 50%

Cette div occupe 50% de son conteneur parent

Largeur 33.33%

Cette div occupe 1/3 (33.33%) de son conteneur parent

Les pourcentages sont relatifs à la valeur du parent. Parfaits pour les mises en page fluides et responsives.

.moitie { width: 50%; }

Résumé des unités CSS

Unité Type Calcul Usage recommandé
px Semi-absolue 1 pixel écran Bordures, espacements précis
pt Absolue 1/72 pouce Impression, documents papier
em Relative police × taille du parent Marges, padding, tailles fluides
rem Relative racine × taille de <html> Tailles de police cohérentes
vw / vh Relative viewport 1% largeur / hauteur écran Mises en page full-screen
% Relative parent % de la valeur du parent Largeurs fluides, responsive
✓ Conseil : Pour une meilleure scalabilité, préférez rem pour les tailles de police et % ou vw pour les largeurs. Utilisez px pour les bordures et les détails mineurs.
