Les feuilles de style

Questions de réflexion :

  • Que se passe-t-il si deux éléments <h1> ont des styles inline différents ? → Chaque élément gardera son propre style unique.
  • Pourquoi cette méthode est-elle déconseillée pour les projets à grande échelle ? → Pas de réutilisabilité, maintenance difficile et code moins lisible.
  • Dans quel cas utilise-t-on le style inline malgré tout ? → Pour des cas ponctuels ou des tests rapides.
