Couleurs et Arrière-plans en CSS

Propriétés de fond disponibles :

1. background-color

Définit la couleur de fond d'un élément. Les valeurs peuvent être exprimées en code hexadécimal (#hex), RGB, ou noms de couleurs.

Exemple : background-color: #FFCC66;

2. background-image

Ajoute une image de fond à un élément. L'image peut se répéter ou rester unique selon les autres propriétés.

Exemple : background-image: url('butterfly.gif');

3. background-repeat

Contrôle la répétition de l'image de fond.

Valeur Comportement
repeat Répète l'image horizontalement et verticalement (défaut)
no-repeat L'image n'est affichée qu'une seule fois
repeat-x Répète l'image horizontalement uniquement
repeat-y Répète l'image verticalement uniquement

4. background-attachment

Détermine si l'image reste fixe ou se déplace avec le contenu.

  • scroll : L'image se déplace avec le contenu (défaut)
  • fixed : L'image reste fixe par rapport à la fenêtre

5. background-position

Positionne l'image de fond. Les valeurs peuvent être des mots-clés (top, left, center, right, bottom) ou des coordonnées (pixels, pourcentages).

Exemples :

  • background-position: top left;
  • background-position: center;
  • background-position: 50% 50%;
  • background-position: 100px 200px;

Propriété raccourcie : background

Vous pouvez combiner toutes les propriétés background en une seule ligne :

background: #FFCC66 url('butterfly.gif') no-repeat fixed right bottom;

Ordre : couleur | image | répétition | attachement | position

Propriétés de couleur de texte :

Propriété color

Définit la couleur du texte. S'applique à tous les éléments de texte.

Exemple : color: #990000;

Résumé des propriétés background

Propriété Valeurs possibles Description
background-color couleur (#hex, rgb, nom) Couleur de fond de l'élément
background-image url('fichier.gif') Image de fond
background-repeat repeat / no-repeat / repeat-x / repeat-y Répétition de l'image
background-attachment scroll / fixed Défilement ou fixe
background-position top left / center / 50% 50% / 100px 200px Position de l'image
