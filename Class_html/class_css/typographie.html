Typographie en CSS

1. Propriétés de police (font-*)

La gestion de la typographie est essentielle pour une bonne présentation visuelle. CSS offre plusieurs propriétés pour contrôler les polices.

font-family : Sélection de la police

Vous pouvez spécifier une liste de polices. Le navigateur utilisera la première disponible.

font-family: Arial, Verdana, sans-serif;

font-size : Taille de la police

Contrôle la taille du texte en diverses unités : px, pt, em, rem, %

font-size: 30px; ou font-size: 1.5em;

font-style : Style de police

Valeurs : normal | italic | oblique

font-weight : Épaisseur de la police

Valeurs : 100, 200... 900 | normal | bold

font-variant : Variante de police

Valeurs : normal | small-caps

line-height : Hauteur de ligne

Une ligne haute améliore la lisibilité (ligne-height: 2em;)

Propriété raccourcie font

Cette ligne utilise la propriété raccourcie font qui combine style, weight, size/line-height et famille

p { font: italic bold 14px/1.8em Arial, sans-serif; }

Ordre : style | variant | weight | size/line-height | family

2. Propriétés de texte (text-*)

text-align : Alignement du texte

Valeurs : left | right | center | justify

text-indent : Retrait de première ligne

Ce paragraphe a un retrait de 50px à la première ligne.

text-decoration : Décoration du texte

Ce texte est souligné (text-decoration: underline;)

Ce texte a une ligne au travers (text-decoration: line-through;)

text-transform : Transformation du texte

Original : ceci devient majuscules

Original : CECI DEVIENT MINUSCULES

letter-spacing : Espacement des lettres

Ce texte a des lettres espacées de 3px

word-spacing : Espacement des mots

Ce texte a des mots espacés de 10px

Résumé des propriétés de police

Propriété Valeurs Description
font-family Arial, Verdana, sans-serif Police à utiliser
font-size 30px, 1.5em, 14pt Taille de la police
font-style normal, italic, oblique Style de la police
font-weight normal, bold, 100-900 Épaisseur de la police
line-height 1.5, 1.8em, 20px Espacement vertical des lignes
text-align left, center, right, justify Alignement du texte
text-decoration underline, line-through, overline Décoration du texte
text-transform uppercase, lowercase, capitalize Transformation du texte
