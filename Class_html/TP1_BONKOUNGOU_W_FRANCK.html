Introduction aux Technologies Web

HTML

HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web.
Il permet de structurer le contenu d'une page en utilisant des balises.
Par exemple, la balise <h1> est utilisée pour les titres principaux, tandis que la balise <p> est utilisée pour les paragraphes.

CSS

CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation d'un document HTML.
Il permet de contrôler la mise en page, les couleurs, les polices et d'autres aspects visuels d'une page web.
Par exemple, la règle CSS color: red; change la couleur du texte en rouge.

JavaScript

JavaScript est un langage de programmation utilisé pour rendre les pages web interactives.
Il permet de manipuler le contenu HTML et CSS, de gérer les événements utilisateur et de créer des applications web dynamiques.
Par exemple, la fonction JavaScript alert("Hello, World!"); affiche une boîte de dialogue avec le message "Hello, World!".
